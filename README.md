# MessMate - Campus Mess Feedback System

A simple tool for VIT (or any college) students to submit anonymous feedback about hostel mess food and generate reports for improvement.

Problem It Solves-

Daily mess food issues in college hostels have no structured tracking. Students vent on groups, but authorities lack data. MessMate fixes this with easy feedback submission and automatic reports.

Features-
- Submit feedback (rating + comments)
- Anonymous option
- Generate average ratings by meal type
- Identify common complaints
- Data saved in CSV (persistent)

Setup & How to Run-
1. Clone the repo: `git clone https://github.com/yourusername/campus-mess-feedback-system.git`
2. Go to folder: `cd campus-mess-feedback-system`
3. Run: `python main.py`



Usage-
- Students: Choose option 1 to submit feedback anytime.
- Mess Committee: Choose option 2 to see reports.

Technology used-
- Python 3
- CSV file handling
- OOP (classes)
- Error handling & validation
- Collections module for analysis
